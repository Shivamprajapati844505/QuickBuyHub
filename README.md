# QuickBuyHub 🛍️

## Description

QuickBuyHub is an online marketplace for branded products, including accessories, clothing, and footwear. It provides a seamless shopping experience with an intuitive user interface, allowing customers to explore a wide range of products, add items to their cart, and make secure transactions.&#x20;

With a powerful admin dashboard, store owners can manage products, track sales, and optimize their business operations. The website also features a responsive design, ensuring a smooth experience across all devices.

## 🚀 Features

- 🔀 **React Router Integration** - Enables seamless navigation.
- 🔍 **Product & Category Search** - Browse products across multiple categories.
- 📌 **Cart Management** - Add, update, and remove items from the cart.
- ⭐ **Rating** -Customer Reviews & Ratings
- 💳 **Payment Integration** - Secure payments using PayPal.
- 🎨 **Modern UI & Responsive Design** - Optimized for all devices.
- 🔐 **User Authentication** - Secure login and registration.
- 🛠️ **Admin Functionality** - Manage products, users, and orders..

## 🛠️ Tech Stack

QuickBuyHub is built with modern web technologies:

- **Frontend:** React.js, React Router, Redux Toolkit, Vite
- **Styling:** Tailwind CSS
- **API Handling:** Fetch/Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Payment Gateway:** PayPal
- **Hosting:** Render (for frontend)

## 📦 Installation & Setup

To set up the project locally, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/Shivamprajapati844505/QuickBuyHub.git
   cd QuickBuyHub
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
   This will launch the application at `http://localhost:3000/`.
