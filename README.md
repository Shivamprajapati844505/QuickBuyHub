# QuickBuyHub 🛒

QuickBuyHub is your go-to destination for branded shopping, offering a seamless and enjoyable shopping experience for Men, Women, Children, and Accessories. With a user-friendly interface, QuickBuyHub ensures that customers can effortlessly explore a wide range of top-brand products including **Nike**, **Adidas**, **Puma**, **Levi's**, and **Zara**. The platform is designed to provide interactive product browsing, efficient cart management, and secure checkout options. Whether you're searching for the latest fashion trends, stylish footwear, or premium accessories, QuickBuyHub has you covered.

##



## 🚀 Features

- 🔀 **React Router Integration** - Enables smooth navigation between different pages.
- 🔍 **Product & Category Search** - Users can easily search for items across multiple categories.
- 📌 **Product Searching & Sorting** - Search and sort products based on keywords like **Men**, **Women**, **Accessories**, **Children**, and **Footwear**. Users can also shop by brand, including **Nike**, **Adidas**, **Puma**, **Levi's**, and **Zara**.
- 📌 **Cart Management** - Allows adding, updating, and removing items from the cart.
- 🎨 **Modern UI & Responsive Design** - Works seamlessly across all devices.
- 🔒 **Secure Authentication** - User accounts and login system.
- 📈 **Dashboard for Admins** - Provides insights and visual reports for administrators.
- 💳 **Online Payment Integration** - Seamless and secure payment gateway for transactions.

## 🛠️ Tech Stack

QuickBuyHub leverages modern web technologies for scalability and performance:

- **Frontend:** React.js, React Router, Redux Toolkit, Vite
- **Library:** ShadCN UI
- **Styling:** CSS Modules / Tailwind
- **API Handling:** Fetch/Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Hosting:** Render

## 🛂 Installation & Setup

To set up the project locally, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/Shivamprajapati844505/QuickBuyHub.git
cd QuickBuyHub
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

This will launch the application at [http://localhost:3000/](http://localhost:3000/).
